BFI 250 Server

The following endpoints are supported:

Films (Public)

  • GET All Films => https://bfi250-node.herokuapp.com/films
  • GET Film By imdbID => https://bfi250-node.herokuapp.com/films/tt0033467
  • - Note: the database id (_id) is the film's imdbID -
  • POST New Film (will soon be admin only)

Ranks (Public)

  • GET All Ranks => https://bfi250-node.herokuapp.com/ranks
  • GET Ranks by bfiSet => https://bfi250-node.herokuapp.com/ranks/bfi1952
  • GET Ranks by imdbID => https://bfi250-node.herokuapp.com/ranks/imdb/tt0040522

Users (Public)

  • POST New User
  • POST Login

Users (Private)

  • POST Logout
  • POST Logout All Devices
  • GET Profile Information
  • PATCH Update Profile
  • DELETE User

View Statuses (Private)

  • GET All viewStatus Information
  • GET viewStatus by ID
  • GET viewStatus by imdbID
  • POST Update to viewStatus
  • DELETE viewStatus by ID
  • DELETE viewStatus by imdbID
