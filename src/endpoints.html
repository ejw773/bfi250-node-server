BFI 250 Server

The following endpoints are supported:

Films (Public)

  • GET All Films => https://bfi250-node.herokuapp.com/films
  • GET Film By imdbID => https://bfi250-node.herokuapp.com/films/tt0033467
  • - Note: the database id (_id) is the film's imdbID -
  • POST New Film (currently disabled due to lack of check for admin rights) => https://bfi250-node.herokuapp.com/films

Ranks (Public)

  • GET All Ranks => https://bfi250-node.herokuapp.com/ranks
  • GET Ranks by bfiSet => https://bfi250-node.herokuapp.com/ranks/bfi1952
  • GET Ranks by imdbID => https://bfi250-node.herokuapp.com/ranks/imdb/tt0040522

Users (Public)

  • POST New User => https://bfi250-node.herokuapp.com/users
  • POST Login => https://bfi250-node.herokuapp.com/users/login

Users (Private)

  • POST Logout => https://bfi250-node.herokuapp.com/logout
  • POST Logout All Devices => https://bfi250-node.herokuapp.com/logoutAll
  • GET Profile Information => https://bfi250-node.herokuapp.com/users/me
  • PATCH Update Profile => https://bfi250-node.herokuapp.com/users/me
  • DELETE User => https://bfi250-node.herokuapp.com/users/me

View Statuses (Private)

  • GET All viewStatus Information => https://bfi250-node.herokuapp.com/viewstatus
  • GET viewStatus by ID => https://bfi250-node.herokuapp.com/viewstatus/61a46f41dfadb8af6b8f0f45
  • GET viewStatus by imdbID => https://bfi250-node.herokuapp.com/viewstatus/film/tt0053472
  • POST Update to viewStatus => https://bfi250-node.herokuapp.com/viewstatus/
  • DELETE viewStatus by ID => https://bfi250-node.herokuapp.com/viewstatus/61a44d9ac76a245c4a97ca59
  • DELETE viewStatus by imdbID => https://bfi250-node.herokuapp.com/viewstatus/film/tt0031885
