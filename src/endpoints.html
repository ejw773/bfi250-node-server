BFI 250 Server

The following endpoints are supported:

Films (Public)

  • GET All Films
  • GET Film By ID
  • GET Film by imdbID
  • POST New Film

Ranks (Public)

  • GET All Ranks
  • GET Ranks by bfiSet
  • GET Ranks by imdbID

Users (Public)

  • POST New User
  • POST Login

Users (Private)

  • POST Logout
  • POST Logout All Devices
  • GET Profile Information
  • PATCH Update Profile
  • DELETE User

View Statuses (Private)

  • GET All viewStatus Information
  • GET viewStatus by ID
  • GET viewStatus by imdbID
  • POST Update to viewStatus
  • DELETE viewStatus by ID
  • DELETE viewStatus by imdbID
